# SaaS Analytics Dashboard
## Project Overview
SaaS Analytics Dashboard is a modern web-based dashboard designed to monitor and analyze important business metrics of a SaaS platform. It provides a clean, responsive, and interactive interface to track revenue, customer growth, sales reports, and performance analytics.
This project simulates a real-world SaaS business environment where users can access key insights through a premium dashboard UI with smooth animations and interactive elements.
## Features
* Interactive Analytics Dashboard
* Customers Management Page
* Reports and Performance Page
* Premium Login Page
* Working Search Bar using JavaScript
* Animated Cards and Graphs
* Responsive Dark Theme UI
* Notification Badge and Hover Effects
## Pages Included
* `index.html` → Main Dashboard
* `customers.html` → Customer Analytics
* `reports.html` → Sales & Performance Reports
* `login.html` → Login Page
## Technologies Used
* HTML5
* CSS3
* JavaScript
* Google Forms (for dummy login/registration system)
## Google Form Integration
Google Form is integrated into the login page to simulate user authentication and data collection without a real backend.
Google Form stores:
* Full Name
* Username
* Email
* Password
* Phone Number
* Feedback / Suggestions
This makes the project more realistic for academic and internship purposes.
